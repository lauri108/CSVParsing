CSVParsing
==========
Parses a CSV file with opening times.

- read-file.php is the main script.
- ravintolat.csv is an example CSV file.
- CSVParser.php and Restaurant.php are helperclasses.
- RestaurantTest.php is a PHPUnit test file.
